Using SMIRP

To make a prediction using SMIRP, enter an RNA sequence into the Sequence field, choose the model which best matches the species you are working with, and hit "Make a prediction". DNA sequences entered into the Sequence field will be converted into RNA sequences by replacing all T characters with U. Due to the computational requirements of generating feature data, the webserver only allows one prediction at a time using pre-computed models. If you wish to predict large datasets, the SMIRP source code is freely available (see below).

About SMIRP

SMIRP is a framework for creating species-specific miRNA prediction systems, leveraging sequence conservation and phylogenetic distance information. Significant improvements in specificity and precision are obtained for non-human hold-out test species when our framework is applied to existing prediction systems. The framework is applicable to all forms of miRNA prediction systems and we expect it to achieve significant improvement in precision and specificity while sustaining sensitivity, independent of the machine learning technique chosen.

About the webserver

The SMIRP web server provides single-miRNA prediction using pre-generated species-specific miRNA prediction models. Prediction is performed using the feature set presented in the HeteroMirPred study [1], and using the LibSVM support vector machine implementation [2]. Currently, six prediction models are available for use on the webserver, representing six distinct species. If you wish to build models specific to a species, you may do so using the SMIRP source code, or request a species-specific model by emailing rpeace@sce.carleton.ca.

SMIRP source code

SMIRP source code is available at https://github.com/jrgreen7/SMIRP/. SMIRP is written in Python, and can be run in Unix environments without installation.

Using the webserver

In order to predict the validity of a potential miRNA sequence, enter the sequence into the above "Sequence" field, select the model species which most closely resembles your target species using the "Model" dropdown menu, and then click the "Make a Prediction" button. Please allow several seconds for the webserver to generate a prediction score for the sequence.
